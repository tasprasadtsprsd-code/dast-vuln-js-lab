A06:2021 · VULNERABLE COMPONENTS

Vulnerable and Outdated Components

Using components with known vulnerabilities — outdated npm packages, jQuery CVEs, prototype pollution in lodash, ReDoS in regex libraries, or exposing package version numbers.

CWE References
CWE-1035, CWE-937, CWE-829
Impact
RCE, data theft, denial of service
